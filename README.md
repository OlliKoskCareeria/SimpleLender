# SimpleLender  
Simple Small-Storage Inventory & Lending App
A lightweight ASP.NET MVC application for tracking small inventory, lending items, and managing statuses like Available, Loaned, or Under Maintenance.
Main user can mark the items status and can easily observe the inventory.
Basic level user can see the available items and own loans including expiery date. 
Email notifications via Gmail can be setted up Via UI. 
Tech.
ASP.NET MVC 5
C#
Entity Framework
SQL Server LocalDB
Bootstrap
JQuery
